# Bangalore Home Price Prediction

A full-stack machine learning web application that predicts residential property prices in Bangalore based on location, area (sqft), BHK, and number of bathrooms.

### 🛠️ Tech Stack

| Layer | Tools Used |
|-------|-----------|
| Data Cleaning & Analysis | Python, Numpy, Pandas |
| Visualization | Matplotlib |
| Model Building | Scikit-learn (Linear Regression) |
| Backend Server | Python Flask |
| Frontend | HTML, CSS, JavaScript |
| IDE | Jupyter Notebook, VS Code, PyCharm |

---

### ✨ Features

- Predict home prices across **200+ Bangalore locations**
- REST API built with Flask serving real-time predictions
- Clean frontend where user inputs sqft, BHK, bathrooms and location
- Full ML pipeline from raw data to deployed model

---

### 📊 ML Concepts Covered

- Data loading and cleaning
- Outlier detection and removal
- Feature engineering
- Dimensionality reduction
- GridSearchCV for hyperparameter tuning
- K-Fold cross validation
- Linear Regression model building

### 🚀 How to Run Locally

**1. Clone the repo**
```bash
git clone https://github.com/Haripriya-M619/Banglore_price_prediction.git
cd Banglore_price_prediction
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Start the Flask server**
```bash
python app.py
```

**4. Open the frontend**

Open `client/index.html` in your browser.
